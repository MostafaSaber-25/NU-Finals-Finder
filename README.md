# 🎓 NU Finals Finder - Spring 2026

A web portal for **Nile University** students to instantly find their final exam schedules, rooms, and times without scrolling through massive spreadsheets.

## 🚀 Features
- **ID Search:** Filter 18,000+ records instantly by Student ID.
- **Live Countdown:** Real-time clock showing time until your next exam.
- **Google Calendar:** One-click integration to save your schedule.
- **Modern UI:** Responsive dark-mode design.

## 🛠️ Tech Stack
- **Backend:** Python (Flask)
- **Frontend:** TypeScript, HTML, CSS
- **Deployment:** Replit
